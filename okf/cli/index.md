# Command-Line Interface

The Loopless CLI provides tools for setup, monitoring, and debugging.

* [Doctor](doctor.md) — Health check and diagnostics
* [Trace](trace.md) — Live execution tracing
* [Replay](replay.md) — Execution trace replay
* [Inspect](inspect.md) — Detailed trace inspection
* [Visualize](visualize.md) — ASCII timeline visualization
* [Init](init.md) — Generate default configuration
